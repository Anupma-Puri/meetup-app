# Meetup App

A full-stack Meetup App built with React, Vite, Bootstrap, and Express.

## Features
- Event listing page
- Search by title and tags
- Filter by online/offline
- Event details page
- Responsive Bootstrap-based UI

## Tech Stack
- Frontend: React, Vite, Bootstrap
- Backend: Express, CORS

## Project Structure
- meetup-frontend
- meetup-backend
